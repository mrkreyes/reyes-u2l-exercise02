Previous PROJECTS

Kunin Mo, Siguro? A Catching Game

Language Used: Java

Kunin Mo, Siguro is a play-until-you-lose game where the goal is to gather as many points as possible to help in Sonnie Robredough’s Angat Buhay Project. There are three stages of the game with different scenes in which the character will collect coins, cash, and gold bars which are converted to the scores. With an initial score of 0 and complete lives consisting of three hearts, Sonnie must avoid items such as Shopee and Lazada parcels, bombs, spikes.

PolySolve Hub: A Mathematical Tool

Language Used: R

The project is an easy-to-use application that combines different mathematical tools. Users can access the main page, where they'll find features like CSV Input, Quadratic Spline Interpolation (QSI), and Polynomial Regression (PR). These features help with things like estimating functions and solving mathematical problems.

Farm2Market Game: A Console Farming Game

Language Used: Python

This is a farm simulation game. You'll play the role of a farmer in this game. You are given an initial amount of PHP. 5,000 pesos as a budget for your farm.
